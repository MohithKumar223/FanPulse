# FanPulse 🏆

**FanPulse** is a high-energy fan engagement platform designed to deepen the tournament experience through gamification, real-time predictions, and AI-driven insights. Built for sports enthusiasts, it transforms passive watching into an interactive, high-stakes competition.

## 🚀 Key Features

- **Gamified Predictions:** Place "votes" on upcoming match winners to earn points and climb the ranks.
- **Mastery Streaks:** Maintain a winning streak to unlock multipliers and climb the "Global Arena" leaderboard faster.
- **AI Match Insights (Gemini Powered):** Leverage Google's Gemini AI to generate tactical previews and high-speed analysis for every match.
- **Personalized Praise:** Receive AI-generated motivating messages based on your specific performance and milestones.
- **Dynamic Leaderboards:** See where you stand against the global fan base in real-time.
- **Progression System:** Earn XP, level up, and unlock unique badges in your "Milestone Vault."

## 🛠️ Tech Stack

- **Frontend:** React 19, TypeScript, Vite
- **Styling:** Tailwind CSS (Modern v4 approach)
- **Animations:** Motion (Framer Motion)
- **Backend:** Firebase (Firestore for real-time data, Firebase Auth for secure Google sign-in)
- **AI Engine:** Google Gemini (Gemini 1.5 Flash via `@google/genai`)
- **Icons:** Lucide React

## 🎯 How It Works

1. **Join the Arena:** Sign in with your Google account to create your fan profile.
2. **Scout matches:** Browse active tournaments like the Premier League or World Cup Qualifiers.
3. **Make your move:** Pick a winner for upcoming matches.
4. **Get the edge:** Use the "AI Insight" button to see what the digital oracle thinks about the matchup.
5. **Track your glory:** Visit your profile to see your win rate, longest streak, and earned achievements.

## 🛡️ Security & Integrity

The platform uses **Firebase Security Rules** to ensure that:
- Users can only modify their own predictions and profile data.
- Tournament and match data are read-only for fans to prevent tampering.
- AI-driven rewards are protected via server-side verification logic simulation.
